# EvenFoundryVTT Wiki

Projects a FoundryVTT D&D 5e session onto Even Realities **G2** AR glasses, driven by **R1** ring gestures.

- 📦 **Install** — see the [README Installation section](https://github.com/Aiacos/EvenFoundryVTT#installation) (Foundry module via manifest URL, bridge via Docker/GHCR, g2-app static host).
- 🧪 **[Testing & Distribution](Testing-and-Distribution)** — how to test on the glasses (dev mode, no expiry) vs. ship to Even Hub (`.ehpk`), and the fix for the **"trial version expired"** error.
- 📖 **Spec** — [`Specs.md`](https://github.com/Aiacos/EvenFoundryVTT/blob/main/Specs.md) (canonical source of truth).
- 🎞️ **Showcase / guide** — <https://aiacos.github.io/EvenFoundryVTT/showcase/>
