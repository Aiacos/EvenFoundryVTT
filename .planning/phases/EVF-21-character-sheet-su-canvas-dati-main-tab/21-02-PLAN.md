<objective>
Extract `ditherTile` and `buildGreyscalePalette` from `raster-worker.ts` (where they are
currently private, unexported, module-level functions) into a new shared
`packages/g2-app/src/raster/dither-utils.ts` that EXPORTS them, then re-import them back
into `raster-worker.ts` with ZERO behavior change.

This is the Wave-0 dependency for the portrait pipeline (Plan 21-04): the canvas sheet
portrait must reuse the exact same Floyd-Steinberg 16-step greyscale dither as the map
tiles, but `raster-worker.ts`'s `ditherTile` is (a) not exported and (b) hard-codes the
module-level `TILE_W`/`TILE_H` constants. The extracted version takes explicit `w`/`h`
params so the 100×60 portrait can reuse it.

Purpose: enables RSHEET-03 (portrait greyscale-dither reuse) without duplicating the algorithm.
Output: new exported dither-utils module + raster-worker delegating to it, zero blast radius
beyond g2-app.
</objective>
