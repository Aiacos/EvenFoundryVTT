```ts
// Concentration is marked via the SPECIAL STATUS EFFECT system, NOT via flags:
//   effect.statuses?.has?.('concentrating')   // ActiveEffect.statuses is a Set
//   OR
//   effect.statuses contains CONFIG.specialStatusEffects.CONCENTRATING (default value = 'concentrating')
//
// Activity metadata lives at: effect.flags?.dnd5e?.activity, effect.flags?.dnd5e?.item
//
// IMPORTANT: Phase 4b SUMMARY noted "Assumption A2" about character-reader using
// item.system.components.concentration. That's the SPELL definition flag (does
// THIS spell require concentration). The cast-spell handler needs the ACTIVE
// EFFECT flag (is THIS actor CURRENTLY concentrating). Both are needed:
//   1. detect that the SPELL being cast requires concentration (item.system.components.concentration === true)
//   2. detect that the ACTOR already has a concentrating ActiveEffect (above)
// Only when BOTH are true do we block the cast.
```
