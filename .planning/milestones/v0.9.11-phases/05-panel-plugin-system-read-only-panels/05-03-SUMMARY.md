## Decisions Made

- **Unicode-safe string ops**: `padRightUnicode` / `truncateUnicode` use `[...str].length` (code-point iteration) not `str.length` (UTF-16 units) — guards against multi-byte glyphs like ◉/○/★/▶ that are common in the panel output
- **Inline dual-edition branching**: `modernRules` boolean tested inline per renderer (single branch `if (snapshot.world.modernRules)`) rather than a strategy object or subclass — keeps each renderer a single function with one mental model
- **Fixture authoring by execution**: Generated fixtures by running `node --import tsx/esm` on real renderers and piping output, not hand-writing ASCII. Guarantees fixture-production alignment at creation time
- **`normaliseRows()` in test helpers**: Per-row `trimEnd()` on both actual and expected sides of fixture comparisons, avoiding false failures from trailing spaces that Biome formatter does not strip from test output

**3. [Rule 1 - Bug] CSTR-FIX-* fixture comparisons failed due to trailing-space mismatch**
- **Found during:** Task 2 (fixture round-trip tests)
- **Issue:** `loadFixture(name).trimEnd()` strips trailing whitespace from all trailing rows (row 17 = 66 spaces becomes empty; then row 16's trailing spaces are also stripped), while `rows.join('\n')` had full trailing spaces on every row
- **Fix:** Introduced `normaliseRows()` helper that applies `trimEnd()` per-row on both sides before comparison
